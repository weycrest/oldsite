Below are some the benefits of VPS over traditional dedicated servers or shared environments.

Root Access - Control your own processes

Inside the virtual machine the "experience" is very similar to a real hardware server. You have "root" level access to a command shell and therefore complete control over system processes, such as web or mail server software or databases. Indeed you can install your own versions of software and libraries. Just like a real hardware dedicated server it is neccessary to keep your VPS updated with latest security patches and bug fixes from the operating system vendor. A Weycrest Root VPS can be maintained using the established backup tools, such as "up2date" "yum" and "apt-get." You can can also use "rpm" within the VPS and compile software from source.

Security

A Xen based VPS from Weycrest offers a secure solution. Your applications are stored on their own virtual "disc" image on a seperate partition. This results in a greater level of file system security than can otherwise be obtained on a normal shared hosting server (especially if the users have shell access). Also your processes have greater protection (isolation) against the crashed applications of other users.
