What is PHP Hosting?

PHP is a widely-used general-purpose scripting language that is especially suited for Web development and can be embedded into your HTML pages. Support for PHP 5 is available on all our Linux shared hosting accounts. PHP is a powerful scripting language allowing you to create dynamic content to your web site.

Getting Started with PHP

It can take some time to get up to speed with PHP if you are not already familiar with a "C" based programming language. It is however possible for relative non programmers to use snippets of PHP in order to maintain their web pages. Let's look at this example of date.php example which "echo's" todays date into a web page:

Most of that page is plain html, but note the PHP tags "<?php" and "?>" embedded into the html to produce this output..

Another useage of PHP for non programmers, is to use it to use the php include function to "stitch" your web pages togethor, in order to create a consistent look and feel throughout your website. In the past, this has been achieved with Server Side Includes (SSI) and even frames, but this can also be done with PHP.

Most web pages consist of a header section, footer, main body and possibly a side bar or two. To maintain more than a few pages using just static html, would be too much work. For instance if you wanted to changer your "header" section you would have to edit all your pages.

Instead you incorporate your "header" section into the top of all your web pages like this:

<?php include ("header.html") ?>

The header.html contains the html code for the top of your website. So if you wish to change the look of your website, you only have one file to edit.
